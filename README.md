# Vulnalyze

**Open-Source Dependency Exploit Propagation Mapper**

Vulnalyzer traces transitive dependency trees across public repositories to identify
projects affected by CVEs, estimates exposure severity, and generates patch-request
issue text for maintainers.

## Architecture

```
vulnalyzer/
├── vulnalyzer/
│   ├── core/           # Shared utilities: DB, version matching, models
│   ├── ingest/         # CVE ingestion from OSV
│   ├── scanner/        # Repository scanning engine
│   │   └── plugins/    # Per-ecosystem manifest parsers (npm, PyPI, Maven …)
│   ├── graph/          # Graph construction, metrics, and export
│   └── api/            # (reserved) FastAPI routes go here
├── scripts/            # CLI entry points
├── tests/
└── data/               # SQLite database lives here
```

## Quick Start

```bash
pip install -r requirements.txt

# 1. Ingest CVEs for a package
python scripts/ingest_cves.py --package lodash --ecosystem npm
python scripts/ingest_cves.py --package django  --ecosystem PyPI

# 2. Scan a single repo
python scripts/scan_repo.py --url https://github.com/timepassuser/vuln_test

# 3. Batch scan from a file
python scripts/batch_scan.py --file repos_to_scan.txt

# 4. Build / refresh the graph
python scripts/build_graph.py

# 5. Query the graph
python scripts/query_graph.py --cve GHSA-35jh-r3h4-6jhm
python scripts/query_graph.py --package lodash --ecosystem npm
python scripts/query_graph.py --repo https://github.com/timepassuser/vuln_test
```

## Graph Model

```
Repository ──USES──> PackageVersion ──INSTANCE_OF──> Package ──AFFECTED_BY──> Vulnerability
Repository ──EXPOSED_TO──> Vulnerability   (derived, for fast queries)
```

Nodes and edges are stored in SQLite (fully queryable without a graph DB).
A JSON export compatible with Obsidian / force-directed frontends is generated by
`build_graph.py`.

## Extending with new ecosystems

Add a file in `vulnalyzer/scanner/plugins/` that implements the `ManifestPlugin`
interface (see `plugins/base.py`).  The scanner auto-discovers all registered plugins.
